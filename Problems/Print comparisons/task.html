Print comparisons

If you have an array of size \(N\), how many comparisons do you need to perform to find each element of this array using jump search?

You are given a number \(N\). You should output \(N\) number of comparisons you need to perform if the element you search for happens to be in this place.


Sample Input:
11

Sample Output:
1 4 3 2 5 4 3 6 5 4 5
